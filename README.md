# quartz-store

Quartz Book Store is a E-commerce website.It is made with Html and CSS.

## Pages ##

Quartz Book Store has following pages

1. Home Page
2. Product Listing Page
     #### Filters by option has three categories : ####
     - Price
     - Category
     - Ratings
3. Cart Management
4. Wishlist Management
5. Authentication
   • Sign-up
   • Login
   • Logout
